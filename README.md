# vanilla-2
A static site rule34.xxx 3rd party wrapper, needs a r34 api key to work. If you aren't aware, r34 is nsfw. You can get an api key [here](https://rule34.xxx/index.php?page=account&s=options) in the API access credentials section (you need a login).

The site doesn't immediately show nsfw content, however by virtue of being a r34 wrapper it does contain nsfw content if it's being used. If you want to check it works without looking at nsfw content, click the middle filter button until it's on safe, and set the likes on the left to 0. If that doesn't work, you can try questionable with a high amount of likes, on the left, however ratings(safe,questionable,explicit) are user submitted and may not be completely accurate.

[link to website](https://sigilquiver.github.io/vanilla-2/)
